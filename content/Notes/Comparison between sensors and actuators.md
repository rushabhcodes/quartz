| **Aspect**         | **Sensor**                                      | **Actuator**                                        |
| ------------------ | ----------------------------------------------- | --------------------------------------------------- |
| **Function**       | Detects and measures physical quantities        | Converts control signals into physical actions      |
| **Role**           | Input device (senses the environment)           | Output device (alters the environment)              |
| **Data Flow**      | Sends data to processor/controller              | Receives commands from processor/controller         |
| **Examples**       | Temperature sensor, motion sensor, light sensor | Motor, relay, LED, heater                           |
| **Purpose**        | Monitor or collect data from surroundings       | Perform an action based on processed data           |
| **Energy Flow**    | Converts physical signal to electrical signal   | Converts electrical signal to physical motion/force |
| **Type of Signal** | Analog or digital signal output                 | Typically requires analog or digital signal input   |
| **Usage**          | Detection, monitoring, feedback                 | Control, response, execution                        |
