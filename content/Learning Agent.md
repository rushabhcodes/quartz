A **Learning Agent** is a type of intelligent agent in artificial intelligence that is capable of learning from its experiences and improving its performance over time. It adapts to its environment by making decisions based on feedback, improving future behavior to achieve better results.

![[Learning Agent.excalidraw.svg]]

### Explanation of Each Component
1. **Performance Element:**
    - Responsible for selecting external actions.
    - Uses current knowledge to make decisions and interact with the environment.    
2. **Learning Element:**
    - Improves the performance element over time.
    - It takes feedback and uses it to update the agent’s knowledge.
3. **Critic:**
    - Evaluates how well the agent is doing based on a fixed performance standard.
    - Provides feedback to the learning element.
4. **Problem Generator:**
    - Suggests exploratory actions that might lead to new and informative experiences.
    - Ensures the agent doesn't get stuck in a local optimum by exploring new options.
